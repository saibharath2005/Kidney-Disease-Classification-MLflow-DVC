How the Classification Works

Our system uses a VGG16 Convolutional Neural Network (CNN) to analyze kidney scan images. The deep learning model extracts patterns from healthy and diseased kidneys and classifies them into categories like Normal, Cyst, Stone, Tumor.

The model was trained using a dataset of labeled kidney images, and its accuracy is validated using test data. The predictions include probabilities for each class, allowing us to understand the confidence of the model.
